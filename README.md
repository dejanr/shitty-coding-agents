# Shitty Coding Agents

Slides and live demo materials for a talk about coding agents, skills, and personal environments.

## Quick start

```bash
direnv allow
```

Then start the presentation:

```bash
demo-it start .
```

## Project layout

- `slides/` – markdown slides
- `demo-it.md` – demo-it presentation script
- `.agents/` – agent-specific assets/skills for demos
